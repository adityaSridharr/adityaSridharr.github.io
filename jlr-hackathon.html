Innovation in Skill Matching and Team Optimization

During my internship at Jaguar Land Rover, I led a diverse team of full-time professionals in a high-pressure 36-hour hackathon. We developed an innovative NLP-powered talent allocation system that transcended the basic "Skill Matching" problem statement by incorporating learning opportunities and deadline optimization. Our solution achieved a remarkable 95% accuracy in task allocation, ultimately winning the global competition.

Technologies Used
  • Natural Language Processing (NLP) for skill extraction and analysis
  • Pre-trained transformer embeddings to balance accuracy and computational feasibility
  • Classification networks for efficient skill matching
  • MLOps pipelines for robust model deployment

  • Optimization algorithms for balancing skill matching with learning opportunities
Challenges Overcome
  • 36-hour time constraint requiring rapid ideation and development

  • Need for production-quality documentation and code in a hackathon setting
  • Creating an intuitive interface for complex allocation decisions

Development Process

Problem Reframing

Instead of treating the challenge as just skill matching, we expanded the scope to create an intelligent talent allocation system that balanced three key factors: skills, learning opportunities, and project deadlines.

Technical Pivot

Facing computational constraints in the 36-hour window, we strategically pivoted from training full LLMs to implementing a classification network using pre-trained transformer embeddings, significantly reducing computational requirements while maintaining high accuracy.

Algorithm Development

Developed optimization algorithms that could weigh different factors in the allocation process, allowing HR managers to prioritize immediate skill matches or long-term employee development based on organizational needs.

UI Design and Implementation

Created an intuitive interface for HR managers to visualize matching results, adjust parameters, and gain insights into allocation decisions.

Presentation and Victory

Successfully presented our solution to international judges, demonstrating both technical excellence and business value, ultimately securing the win against global competition.
